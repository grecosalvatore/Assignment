RESOURCE ORGANIZATION

RESOURCE VERB REQUEST BODY STATUS RESPONSE BODY MEANING
rns GET - 200 OK Rns Return link to all main resources in the sytem (Entry Point of the System)
rns/places GET - 200 OK Places List of places in the system
rns/places/roadSegments GET - 200 OK Places List of road segments in the system
rns/places/parkingAreas GET - 200 OK Places List of parking areas in the system
rns/places/gates GET - 200 OK Places List of gates in the system
rns/places/{id} GET - 200 OK Place Read place with placeId={id} (can be every type of place)
404 Not Found - The place is not in the system
rns /places/{id}/currentVehicles GET - 200 OK Vehicles Read the set of vehicles that are now in the selected place
404 Not Found - The place is not in the system
rns/vehicles GET - 200 OK Vehicles Read the set of all vehicles in the system
rns/vehicles/{id} GET - 200 OK vehicle Read single vehicle with plateId={id}
404 Not Found - The vehicle is not in the system
rns/vehicles/{id} PUT vehicle 200 OK suggestedPath Permission granted for the vehicle with plateId={id} to enter the system, the suggested path is returned
400 Bad Request - Permission not granted - origin or destiation are not in the system or validation of request body error
403 Forbidden - Permission not granted - Origin is not an IN/OUT gate or the vehicle is already in the system
rns/vehicles/{id} DELETE - 200 OK - Administrator succesfully remove the vehicle from the system
403 Forbidden - Only administrator can delete vehicle from the system (NOT IMPLEMENTED)
404 Not Found - The vehicle is not in the system
rns/vehicles/{id}/currentPosition GET - 200 OK position Read information about current position for the {id} vehicle
404 Not Found - The vehicle is not in the system
rns/vehicles/{id}/currentPosition PUT position 200 OK suggestedPath Permission granted to change position (in the request) for the {id} vehicle and the suggestedPath is returned
400 Bad Request - Permission not granted - Valildation error or new position is not in the system
403 Forbidden - Permission not granted - New position is not reachable from current position
404 Not Found - The vehicle is not in the system
rns/vehicles/{id}/currentPosition DELETE - 200 OK - Permission granted for the vehicle to exit the system from this current position
403 Forbidden - Permission not granted - current position is not OUT/INOUT gate
404 Not Found - The vehicle is not in the system
rns/vehicles/{id}/state GET - 200 OK state Read current state of the vehicle
404 Not Found - The vehicle is not in the system
rns/vehicles/{id}/state PUT state 200 OK state Vehicle succesfully changed the state
400 Bad Request - Body validation error
404 Not Found - The vehicle is not in the system
rns/vehicles/{id}/state GET - 200 OK suggestedPath Read the suggestedPath for the vehicle
404 Not Found - The vehicle is not in the system
